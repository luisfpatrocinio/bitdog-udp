# bitdog-udp

UDP sender for Raspberry Pi Pico W.  
Sends packets via Wi-Fi when buttons are pressed.  
Shows status on an OLED display.

## Setup

Edit `main.c` with your Wi-Fi and target IP, then compile and flash.
